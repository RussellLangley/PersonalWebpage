The primary goal of this page is to display the courses I have completed at Algonquin, or am currently taking, and provide a brief overview of them.







CST8284

Object Oriented Programming (Java)

Explores object oriented programming methods using Java and builds upon concepts learned from 8116. Things like abstraction and encapsulation, to name just two examples, are covered and used in actual laboratory assignments.

CST8285

Web Programming

Introduces website programming using JavaScript, HTML5, PHP, and CSS and uses those tools to build projects and websites.

CST8285

Operating Systems Fundamentals (GNU/Linux)

Explores fundamental concepts in operating systems, focusing on GNU and Linux platforms and the syntax used to operate them within the terminal.

CST2355

Database Systems

Builds upon what was learned from CST8215 and introduces Oracle, Access, and Microsoft SQL Server Managemnt Studio. Advanced SQL, rollup and cube operations, as well as metadata manipulation are some primary topics covered.

MAT8001

Tech Math for Computer Science

Modudles on Algebraic Operations and Algebra Basics, Computer Number Systems, Boolean Algebra, Factoring and Solving Equations, Trigonometry, Graphing, Vectors, and Systems and Linear Equations are covered.

CST8101

Computer Essentials

Introduces essential information on the use of computer software, hardware, and its management.

CST8116

Introduction to Computer Programming

Provides a broad overview of object oriented programming, focusing on Java. Topics such as sequential structures, selection structures, repitition structures, variables, constants, methods, and many more are covered and practiced.

CST8215

Introduction to Database

Introduces students to the use of databases in personal and professional use. Databases are created, modifed, and queried using DBMS's.

CST8300

Achieving Success in Changing Environments

Primarily comprised of group work projects and writing assignments, this course functions to prepare students for the work force and professional life. Soft skills are seldom practiced and thus this course functions to emphasize those skills and build upon them.
